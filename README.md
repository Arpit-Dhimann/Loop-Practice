# Loop-Practice

This repository is dedicated to mastering **Loops and Iterations** in Java. This project covers the fundamentals of iterative logic, ranging from simple counters to complex pattern generation.

## 🚀 Projects Included

### For Loops
- **For_Loop_Basic.java**: Simple iteration to understand counting logic.
- **For_Loop_EvenOdd.java**: Dynamic range checker using conditional logic within loops.

### While Loops
- **While_Loop_Countdown.java**: User-defined countdown timer implementation.
- **While_Loop_Sum.java**: Mathematical summation logic using accumulation.

### Do-While Loops
- **Do_While_Menu.java**: Interactive menu-driven system.
- **Do_While_Password.java**: Secure input validation mechanism.

## 🛠️ Concepts Covered
- **Iterative Logic**: Implementing `for`, `while`, and `do-while` constructs.
- **Input Handling**: Using `java.util.Scanner` for dynamic, user-interactive programs.
- **Defensive Programming**: Ensuring memory safety with `sc.close()` and robust condition management.
